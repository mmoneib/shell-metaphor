# shell-metaphor
## Description
A collection of simulations and models with the aim to explore different aspects of life in mathematical, stochastic, or poetical way.
## Motivation
A testbed of concepts and thought experiments. Shell Scripts are used here as a prototyping language with the intent to quickly test and demonstrate a theoretical idea or concept. More advanced development or exploration would be done, if needed, in a separate repository with a more advanced language.
## Tags
ideation,experimentation,conceptual,modeling,theory,philosophy.
## Metaphors
Each "metaphor" corresponds to a shell script in the repository.
### Coincidence
What is a coincidence and how likely one can happen? This script reduces the phenomenon into a configurable setup of two independent agents with specic movement strategies.
### Condorcet Paradox
An implementation of the famous paradox, which shows the divergene between the elections reusults and the popularity of the candidateds in a representatie democracy.
